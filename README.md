# TRIPLE REPORT

店舗結果・レポート確認用のWebアプリです。

## 開発

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

## デプロイ（GitHub Pages）

1. リポジトリの **Settings** → **Pages** を開く
2. **Build and deployment** の **Source** で **GitHub Actions** を選択
3. `main` ブランチへ push すると自動でビルド・デプロイされます

公開URL: `https://suuu31djdrm-a11y.github.io/triple-report/`
